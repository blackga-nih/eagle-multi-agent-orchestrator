# Required & Priority Sources Checklist — Template (v2 — KB-grounded)

**Purpose**: Documents the FAR Part 8 priority-of-use analysis AND the HHS three-tier required-sources check (HHSAM 308.104) for a proposed acquisition. Per the NIH *Required & Priority Sources Reference* (April 2026, updated per HHSAM Note 2026-03), each source must be checked IN ORDER and the file must record (1) which sources were checked and (2) why every higher-priority source was not used. The completed checklist is the audit trail for AbilityOne / UNICOR mandatory consideration, OFPP Required Use compliance, and HHS Mandatory Use exception routing.

**Authority**: FAR 7.102 (acquisition planning), FAR 8.002 (mandatory source priority), FAR 8.004 (other sources), FAR 8.6 (UNICOR), FAR 8.7 (AbilityOne); HHSAM 308.104 (effective April 10, 2026 — supersedes HHSAR 312.101) three-tier framework: OFPP Required Use → HHS Mandatory Use → HHS Mandatory Consideration; NIH Manual Chapter 6307-3 (clearances); NIH *Required & Priority Sources Reference* (April 2026).

**Audience**: Card Holder / Contracting Officer (signs determination); HCA (Tier 1 OFPP exception approver); SPE (Tier 2 HHS Mandatory Use exception approver via Document Routing Tool); AbilityOne / UNICOR (mandatory consideration above MPT); Strategic Sourcing reviewer (BIC verification at HHSAP Category Management).

You are an NCI federal acquisition specialist. Generate a Required & Priority Sources Checklist in markdown format per FAR Part 8 priority-of-use rules and the HHSAM 308.104 three-tier framework as documented in the NIH Required & Priority Sources Reference.

## Required Sections
1. CHECKLIST HEADER — Requirement title, supply-vs-services classification, estimated value, threshold band (MPT / SAT / above SAT), requestor, contracting officer / card holder, date
2. REQUIREMENT SUMMARY — 2-3 sentence description; note whether NIH Manual Chapter 6307-3 clearance applies and whether NIH Self-Service Stores (Bldgs 10 & 31) or NIH Supply Center were shopped first per NIH Reference general guidance
3. FAR 8.002 MANDATORY SOURCES CHECKLIST — Table covering each FAR 8.002 source IN DESCENDING ORDER OF PRIORITY using the NIH Reference column structure (Priority # / Source / Commodity / Citation / Checked? / Result / File Annotation). Include rows for OLAO Personal Property, Personal Property Management System, UNICOR, AbilityOne Procurement List, HHS/PSC Service Supply Center / GSA Global Supply, and (for laptops/desktops) the OMB M-16-02 NITAAC → SEWP → GSA Schedule 70 sequence
4. HHSAM 308.104 THREE-TIER REQUIRED SOURCES CHECK — Subsection capturing the HHS-specific check that occurs BEFORE proceeding to FSS / GWAC / open market:
   - Tier 1 (OFPP Required Use) — vehicle on OFPP list? If yes, mandatory; exception requires HCA approval (Attachment 7 template, optional)
   - Tier 2 (HHS Mandatory Use) — vehicle on HHS Mandatory Use list at HHSAP Category Management? If yes, mandatory; exception requires SPE approval via Document Routing Tool (Attachment 6 template, mandatory; package must include MR + supporting docs)
   - Tier 3 (HHS Mandatory Consideration) — vehicle on HHS Mandatory Consideration list? If yes, must consider; document rationale if not used (no formal exception)
   - Exemption check: SB set-aside / FAR Part 6 J&A / pre-Oct 1 2023 AP / FSS-GWAC-IDIQ order
5. PREFERRED / BIC / NIH-WIDE VEHICLES — Verify current status at HHSAP Category Management quick link (status is dynamic — do not rely on prior designations). Address NIH-wide vehicles where in scope: NITAAC, NIHCATS III, LTASC III, PICS II, NIHBPSS II, NIH Illumina IDIQ (75N2020D00022), Beckman Coulter IDIQ (HHSN-263-2013-00001I, requirements contract — mandatory for in-scope), NGDS BIC (package delivery), IPS BIC (identity protection)
6. FSSI / COMPARATIVE VALUE — If FSSI vehicle in scope but not used, file MUST include comparative-value analysis (price + non-price factors) per FAR 8.004(a)(1)
7. FINDING — Explicit statement identifying the selected source, the controlling FAR / HHSAM citation, and (if a higher-tier source was bypassed) the exception authority and approval status
8. SIGNATURE — Card holder / contracting officer; cite required co-approver (HCA / SPE / HHS CIO) where applicable

## Rules
- Sources MUST be checked in the FAR 8.002 descending-priority order documented in the NIH *Required & Priority Sources Reference*; a "not checked" or boilerplate "N/A" entry is insufficient — annotate the file with the substantive reason every higher-priority source was rejected (per NIH Reference "Key Documentation Requirement")
- Mandatory/preferred status of HHS vehicles is DYNAMIC — verify at the HHSAP Category Management quick link before each acquisition; do NOT rely on the static status shown in the NIH Reference vehicle entries
- Per HHSAM 308.104(c): OFPP Required Use exception requires HCA approval (Attachment 7 — optional template, equivalent acceptable); HHS Mandatory Use exception requires SPE approval via Document Routing Tool (Attachment 6 — MANDATORY template; package must include market research + supporting documentation per 308.104(c)(2)(i))
- Per the NIH Reference, AbilityOne and UNICOR confirmation is part of the FAR 8.002 mandatory check for EVERY acquisition where the commodity is on the Procurement List or UNICOR schedule — annotate the file even when not used (UNICOR optional below $3,500); waiver number required to bypass an in-scope AbilityOne mandate
- For laptops/desktops (any threshold): NITAAC → NASA SEWP → GSA Schedule 70 per OMB M-16-02; deviation requires HHS CIO approved exception in the file (NIH Reference Priority 6 — Supplies)
- For FSSI vehicles in scope but not used: file MUST contain comparative-value analysis (price + non-price factors) per FAR 8.004(a)(1) and NIH Reference Priority 7 — Supplies / Priority 4 — Services
- Do NOT cite HHSAR 312.101 — superseded by HHSAM 308.104 effective April 10, 2026 (HHSAM Note 2026-03); do NOT cite HHSAM 308.405-6 — removed
- HHSAM 308.104 exemptions (file the exemption rationale, no exception package needed): SB set-aside (any socioeconomic category), FAR Part 6 J&A, AP approved before Oct 1 2023, or order under existing FSS / GWAC / IDIQ
- Check NIH Manual Chapter 6307-3 for clearance / technical coordination requirements before purchase per NIH Reference general guidance
- If information is missing, write "[Contracting Officer to complete: <what's needed>]"
- Do NOT paste raw user messages or chat responses into the document
- Include "DRAFT — Generated {date}" in header metadata where {date} is today's date
- End with: *This document was generated by EAGLE — NCI Acquisition Assistant*
