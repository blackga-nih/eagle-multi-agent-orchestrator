## Deployment Steps (2026-02-27)

### Pre-flight (run from Windows PowerShell or Git Bash)
```bash
# Confirm correct account/region
aws sts get-caller-identity
# Expected: Account: 274487662938, Region: us-east-1

# Validate template
aws cloudformation validate-template \
  --template-body file://aws/cloud_formation/ec2.yml
```

### Deploy EC2 Stack
```bash
aws cloudformation deploy \
  --template-file aws/cloud_formation/ec2.yml \
  --stack-name eagle-ec2-dev \
  --parameter-overrides file://aws/cloud_formation/params/dev/ec2.json \
  --capabilities CAPABILITY_NAMED_IAM \
  --region us-east-1
```

### Get Connection Info
```bash
# Instance ID
INSTANCE_ID=$(aws cloudformation describe-stacks --stack-name eagle-ec2-dev \
  --query 'Stacks[0].Outputs[?OutputKey==`InstanceId`].OutputValue' --output text)

# Private IP
IP=$(aws cloudformation describe-stacks --stack-name eagle-ec2-dev \
  --query 'Stacks[0].Outputs[?OutputKey==`PrivateIp`].OutputValue' --output text)

echo "Instance: $INSTANCE_ID  IP: $IP"
```

### Connect via SSM Session Manager (Recommended â€” no SSH key needed)
```bash
# Install Session Manager plugin on Windows:
# https://docs.aws.amazon.com/systems-manager/latest/userguide/session-manager-working-with-install-plugin.html

aws ssm start-session --target $INSTANCE_ID --region us-east-1
```

### Connect via SSH (alternative)
```bash
# Get key from SSM Parameter Store
KEY_ID=$(aws cloudformation describe-stacks --stack-name eagle-ec2-dev \
  --query 'Stacks[0].Outputs[?OutputKey==`KeyPairId`].OutputValue' --output text)

aws ssm get-parameter \
  --name "/ec2/keypair/${KEY_ID}" \
  --with-decryption \
  --query 'Parameter.Value' \
  --output text > ~/.ssh/eagle-runner-dev.pem

chmod 400 ~/.ssh/eagle-runner-dev.pem
ssh -i ~/.ssh/eagle-runner-dev.pem ec2-user@${IP}
```
