# Cross-Domain Learnings

> Shared knowledge log. When a domain's `self-improve` discovers something that affects
> multiple expert areas, it appends here. `parallel_expert_self_improve` reads this first
> so all domain agents start with cross-cutting context.

---

## Format

```markdown
## YYYY-MM-DD — [short title]
- **Discovered by**: `{domain}` expert during session {short-id or description}
- **Affects**: `{domain1}`, `{domain2}`, ...
- **Learning**: {concrete, specific finding}
- **Action taken**: Updated `{domain}/expertise.md`. Domains `{others}` need review.
```
